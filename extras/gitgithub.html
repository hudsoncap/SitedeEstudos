Comandos terminal Git

  • git config --global user.name "Hudson André dos Reis Cardoso": configura o nome do Usuário
  • git config --global user.email "hud.cardoso@hotmail.com" :Configura seu Email
  • git config --global core.editor 'code --wait' :Define o vscode como padrão.
  • git config --global -e:Checar qual é o editor padrão.
  • git config --list:Retorna todas configurações do git
  • mkdir "nomedoprojeto": Cria uma Pasta.
  • cd "nome da pasta": Entra na pasta.
  • gir init: inicializa o repositorio e observa mudanças.
  • ls: Verifica as pastas na pasta.
  • cd ..: volta para a pasta anterior.
  • gid add (arquivo)Adiciona o Arquivo.
  • git statusrelata o status do arquivo.
  • git commit -m "Add (nome do arquivo)"criando uma versão do arquivo
  • git log: Mostra o Histórico do Log.
  • git log --decorateMostra o log com algumas informações adicionais
  • git log --author="hudson": Mostra todos os logs desse autor
  • git shortlog: mostra a lista de commit de usuarios
  • git shortlog -snmostra a lista com o nome e a quantidade de logs dos usuários
  • git log --graph: mostra em forma grafica o que está acontecendo com os brants
  • git show "rash commit" verifica o que está acontecendo com essa commit
  • git diff: Mostra as mudanças no arquivo log do git antes de commitar.
  • git diff --name-only: Mostra os nomes dos arquivos modificados antes de commitar
  • git commit -am "nome da modificação"commita o arquivo
  • git checkout (nome do arquivo)reseta o arquivo a versão anterior antes de ser modificado.
  • git reset HEAD (nome do arquivo): Reseta o arquivo ao estado de "modificado" mesmo depois de ter sido adicionado a modificação.
  • git reset --soft (hash): vai pegar as modificaçoes reseta o commit, mas o arquivo volta pra stated, mas pronto pra ser comitado novamente.
  • git reset --mixed (hash): vai pegar as modificaçoes reseta o commit, mas o arquivo volta para modifited.
  • git reset --hard (hash): vai pegar as modificaçoes reseta o commit, e tudo o que foi modificado no arquivo.

Editando arquivo: no (VIM)

  • vi "nomedoArquivo": Edita o Arquivo.
  • i:entra no modo insert(libera digitação)
  • esq sai do comando anterior
  • ;indica que vai iniciar outro comando
  • wqescreve Salva e sai.

GitHub Remote:

  • git remote add origin https://github.com/hudsoncap/(nome do arquivo) envia o arquivo para o github remote(origin)
  • git remotemostra se ja existe um repositorion origin
  • git remote -vmonstra mais detalhes existentes na origin
  • git push -u origin mainenvia todo os arquivos e todos os logs para o repositório determinado (origin onde vai) main(de onde vem)
  • git push origin main(branch) commita tudo para o github remoto(origin)
