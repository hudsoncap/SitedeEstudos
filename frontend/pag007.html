Tabela em HTML:

Hierarquia de tabela HTML:

        TABLE = tabela
             TABLE ROW = linha de tabela 
                 TABLE HEADER = cabeçalho de tabela 
                    TABLE DATA = dado de tabela.

Tags de tabela:

  • table (tabela) > tr (linha) > td (dados) >

Alinhamento horizontal e vertical os textos:

Horizontal: text-align: left,right e center

Vertical: vertical-align: top, bottom, center

Unir a tabela use:

border-collapse: collapse

Tabelas Grandes:

Anatomia para Tabelas Grandes:

Table > thead(tr,td,th) > tbody(tr,td,th) tfoot(tr,td,th)

Exemplo:


<table>
  <caption>POPULAÇÃO das Unidades Federativas
    <thead>
        <tr>
          <th scope="col"> Estado
          <th scope="col"> População
       </tr>
    </thead>
    <tbody>
        <tr>
          <td>Paraná
          <td class="numb">12546456
        </tr>
        <tr> 
          <td>Rio de Janeiro
          <td class="numb">89455451
        </tr>
        <tr>
            <td>São Paulo
            <td class="numb">945456544566
       </tr>
    </tbody>
    <tfoot>
        <tr>
           <th scope="row">Total populacional
           <td>xxxxxxxxxxx
       </tr>
    </tfoot>

</table>
