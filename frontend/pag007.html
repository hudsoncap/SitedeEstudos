Tabela em HTML:

Hierarquia de tabela HTML:

        TABLE = tabela
             TABLE ROW = linha de tabela 
                 TABLE HEADER = cabeçalho de tabela 
                    TABLE DATA = dado de tabela.

Tags de tabela:

  • table (tabela) > tr (linha) > td (dados) >

Alinhamento horizontal e vertical os textos:

Horizontal: text-align: left,right e center

Vertical: vertical-align: top, bottom, center

Unir a tabela use:

border-collapse: collapse

Tabelas Grandes:

Anatomia para Tabelas Grandes:

Table > thead(tr,td,th) > tbody(tr,td,th) tfoot(tr,td,th)

considerações:

  • Usar th(titulos)
  • Usar td(dados)
  • scope="col"(titulos de linhas ↓ )
  • scolpe="row"(titulos de colunas → )

    • PS:scolpe é ao contrario de span

  • scolpe="colgroup"(titulos de linhas ↓ com mais de um span)
  • scolpe="rowgroup"(titulos de colunas → com mais de um span)

    • Mesclagem de tabelas

  • <td colspan="2">(td ocupara "2" COLUNAS → )
  • <td rowspan="3">(td ocupara "3" LINHAS ↓ )

    • Aplicando CSS em Colunas row esclusiva

  • <colgroup> <col class="nome" > </colgroup>
  • <colgroup> <col class="nome" span='2'>(span espande o css para '2 colunas') </colgroup>
  • Para deixar a tabela responsiva a tela, envolope a tabela por uma <div> e no css use overflow-x:auto;
