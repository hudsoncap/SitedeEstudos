Map e reduce

Map

map: Percorre uma coleção e gera valores alterados dos seus itens.

map é melhor que while e if por que ele retorna uma nova lista, não altera a lista atual.


    const produtos= [
        { nome: "nootbook",
            preco: 2100}, 
        { nome: "celular",
            preco: 400}, 
    ]                   
    const precoemreal   = produtos.map(p => p.preco *4)
    console.log(precoemreal)

o Console irá retornar o preco do produto * 4: [8400, 1600]

reduce

Reduce: Percorre uma coleção e gera alguma informação dela ex: Soma de valores de todos produtos.

reduce é melhor que while e if por que ele retorna uma nova lista, não altera a lista atual.

    const produtos= [
    { nome: "nootbook",
    preco: 2100},
    { nome: "celular",
    preco: 400},
        ]
            const total   = produtos.reduce((a,b) => a+b.preco , 0)
            console.log(total)


o Console irá retornar o preco de todos produtos somados: [2500]

também podemos usar map e reduce juntos.


const produtos= [
    { nome: "nootbook",
    preco: 2100}, 
    { nome: "celular",
    preco: 400}, 
    ]                   
const total   = produtos.map(p => p.preco * 4).reduce((a,b) => a+b,0)
console.log(total)


o Console irá retornar o preco de todos produtos somados e convertidos: [10000]
