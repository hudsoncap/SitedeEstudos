Comandos

  • git config --global user.name "Hudson André dos Reis Cardoso": configura o nome do Usuário
  • git config --global user.email "hud.cardoso@hotmail.com" :Configura seu Email
  • git config --global core.editor 'code --wait' :Define o vscode como padrão.
  • git config --global -e:Checar qual é o editor padrão.
  • git config --list:Retorna todas configurações do git
  • mkdir "nomedoprojeto": Cria uma Pasta.
  • cd "nome da pasta": Entra na pasta.
  • gir init: inicializa o repositorio e observa mudanças.
  • ls: Verifica as pastas na pasta.
  • cd ..: volta para a pasta anterior.

Editando arquivo: no (VIM)

  • vi "nomedoArquivo": Edita o Arquivo.
  • i:entra no modo insert(libera digitação)
  • esq sai do comando anterior
  • ;indica que vai iniciar outro comando
  • wqescreve Salva e sai.
